# API Reference

Base URL: `/api/v1`. Interactive OpenAPI docs are generated by FastAPI at **`/api/v1/docs`** (schema at `/api/v1/openapi.json`).

Conventions: camelCase JSON; errors always use `{"error": {"code", "message", "details?", "requestId"}}`; bearer JWT (Supabase) where marked; admin routes re-verify `profiles.role` in the database.

## Health
| Method & path | Auth | Purpose |
|---|---|---|
| `GET /health` | — | Liveness (name, version, environment) |
| `GET /readiness` | — | Classifier-config + database checks |

## Analyses
| Method & path | Auth | Purpose |
|---|---|---|
| `POST /analyses/preview-quality` | optional | Quality gate only; 200 with `acceptable` true/false, envelope for structural failures. Rate-limited. |
| `POST /analyses` | optional | Full analysis (multipart `image`, `save_image`, `questionnaire` JSON). Guests: full result, nothing persisted. Users: persisted; image stored only on opt-in. Rate-limited. |
| `GET /analyses?page=&page_size=` | user | Paginated history summaries |
| `GET /analyses/{id}` | owner | Full detail + short-lived signed image URL when stored |
| `DELETE /analyses/{id}` | owner | Delete analysis (+ stored image) |
| `POST /analyses/{id}/save-image` | owner | Store the photo post-hoc (client re-sends it; validated like an analysis upload). Rate-limited. |
| `DELETE /analyses/{id}/image` | owner | Delete only the stored image |
| `GET /analyses/{id}/palette` | owner | Palette resolved for the analysis (season + sub-season) |
| `GET /analyses/{id}/recommended-products` | owner | CIEDE2000-ranked products with match scores |

## Palettes & products (public catalogue)
| Method & path | Auth | Purpose |
|---|---|---|
| `GET /seasons` | — | 4 seasons + 12 sub-seasons |
| `GET /seasons/{slug}?subseason=` | optional | Grouped palette + cosmetics; favourite flags when authenticated |
| `GET /products?…` | — | Directory: `category`, `gender`, `season`, `store`, `q`, `sort` (newest/price_asc/price_desc/name), `page`, `page_size` |
| `GET /products/{id}` | — | Product detail |

## Favourites & account
| Method & path | Auth | Purpose |
|---|---|---|
| `POST·DELETE /colours/{id}/favourite` | user | Favourite palette colours |
| `GET /me/favourite-colours` | user | List favourite colours |
| `POST·DELETE /products/{id}/favourite` | user | Favourite products |
| `GET /me/favourite-products` | user | List favourite products |
| `GET /me` · `PATCH /me` | user | Profile (display name) |
| `GET·PATCH /me/preferences` | user | `defaultImageStorage` (default **false**), currency, reduced motion |
| `POST /me/consents` | user | Append-only consent events |
| `DELETE /me/analyses` | user | Wipe complete history (+ stored images) |
| `DELETE /me` | user | Delete account — cascades every owned row and storage object |

## Admin (role verified server-side; mutations audit-logged)
| Method & path | Purpose |
|---|---|
| `GET /admin/stats` | Anonymised aggregates only |
| `GET /admin/audit-logs?page=` | Append-only audit trail |
| `GET /admin/algorithm-versions` | Classifier versions |
| `GET /admin/settings` · `PUT /admin/settings/{key}` | Runtime settings |
| `GET·POST /admin/stores` · `PATCH /admin/stores/{id}` | Store CRUD/activation |
| `GET·POST /admin/products` · `PATCH /admin/products/{id}` | Product CRUD/activation |
| `POST /admin/palette-colours` · `PATCH·DELETE /admin/palette-colours/{id}` | Palette colour CRUD |
| `PATCH /admin/seasons/{slug}` | Season text/activation |
| `POST /admin/cosmetics` · `PATCH·DELETE /admin/cosmetics/{id}` | Cosmetic CRUD |
| `POST /admin/products/import?dry_run=` | CSV import (dry-run default) |
| `GET /admin/products/imports` · `GET /admin/products/imports/{id}/errors` | Import history + row errors |
